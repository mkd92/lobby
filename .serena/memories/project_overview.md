# Project Overview

**Name:** Lobby
**Purpose:** Mobile-optimized property management system for managing rental properties, hostels, tenants (customers), leases, and payments. Includes PWA support.

**Tech Stack:**
- React 19 + TypeScript
- Vite 7 (bundler + dev server)
- React Router v7 (client-side routing)
- Supabase (backend: auth, database)
- vite-plugin-pwa (PWA support)
- ESLint + typescript-eslint (linting)

**Backend:**
- Supabase (hosted or local via Docker)
- Local Studio: http://127.0.0.1:54323
- Local API: http://127.0.0.1:54321
- Credentials in `.env` (local) and `.env.local`

**Deployment:** Vercel (`vercel.json` present)
